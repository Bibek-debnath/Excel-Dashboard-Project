Problem Statement: Ferns and Petals Sales Analysis
 You have been given a dataset from FNP (Ferns and Petals) that specializes in sending gifts for
 various occasions like Diwali, Raksha Bandhan, Holi, Valentine's Day, Birthdays, and
 Anniversaries. The dataset contains details about the products, orders, customers, and relevant
 dates. Your task is to analyze this dataset to uncover key insights related to sales trends,
 customer behavior, and product performance.
The dataset used in this project contains complete order-level details, which allow deep analysis of customer and sales behavior. The dataset includes:

Order details such as order quantity and order dates

Product information, including product category and product name

Revenue and customer spending details

Delivery dates, which help measure delivery performance

City-level data, showing where orders are coming from

Occasion type, such as festivals or personal celebrations

By analyzing this data, we can clearly understand:

What customers are buying

When they are buying

Which products generate the most revenue

Which cities contribute the most orders
